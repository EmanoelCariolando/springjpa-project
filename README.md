# Spring JPA Project

Este é um projeto simples desenvolvido para **estudo e prática de Spring Boot com JPA (Hibernate)**.  
O objetivo é entender como funciona a integração entre o Spring e o banco de dados, incluindo o mapeamento de entidades, repositórios e operações básicas de CRUD.

## Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Banco de dados H2 (em memória)
- Maven
